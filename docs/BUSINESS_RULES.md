## Plataforma

BR-001

Cada Usuario pertenece a un único Rol del sistema.

BR-002

Solo existe un SuperAdministrador global.

BR-003

Todo Usuario, excepto el SuperAdministrador, pertenece a una Congregación.

BR-004

Los Roles determinan únicamente los permisos dentro del sistema.

Los Roles no representan responsabilidades dentro de una congregación.


## Congregacion

BR-100

Toda la información pertenece a una Congregación.

BR-101

Nunca se comparte información entre Congregaciones.

BR-102

Un Publicador pertenece únicamente a una Congregación.

BR-103

Un Publicador puede tener múltiples Privilegios.

BR-104

Los Privilegios son administrables mediante un catálogo.

BR-105

Los Grupos de Servicio pertenecen a una Congregación.

BR-106

Un Orador Visitante no pertenece a la Congregación anfitriona.


## Programacion

BR-200

Todo Programa Oficial proviene de una fuente externa compatible.

BR-201

El Programa Oficial nunca se modifica.

BR-202

Todo Programa Local proviene de un Programa Oficial.

BR-203

Las modificaciones solo se realizan sobre el Programa Local.

BR-204

Todo Evento pertenece a un Programa.

BR-205

Todo Evento tiene un Tipo de Evento.

BR-206

Cada Parte pertenece a un Evento.

BR-207

Una Parte puede tener una o varias Asignaciones.

BR-208

Una Asignación puede tener un único Asignado.

BR-209

Todo Programa Local puede exportarse a PDF.

BR-210

El sistema debe permitir reemplazar total o parcialmente un Programa Local cuando existan circunstancias especiales.

Ejemplos:

- Visita del Superintendente de Circuito.

- Asamblea.

- Conmemoración.

- Programa Especial.


## Ministerio

BR-300

Un Exhibidor pertenece a una Congregación.

BR-301

Un Lugar puede tener varios Horarios.

BR-302

Cada Horario puede tener varios Turnos.

BR-303

Cada Turno tiene exactamente dos Asignados.

BR-304

Los Turnos pertenecen a un Lugar.

# Agent Rules

Antes de modificar código:

- Leer AI_CONTEXT.md.
- Leer ARCHITECTURE.md.
- Explicar el plan.
- Esperar aprobación si modifica más de 5 archivos.

Después de modificar:

- Ejecutar pruebas.
- Explicar los cambios.
- Indicar riesgos.
- Sugerir actualización de documentación si aplica.

Nunca:

- Cambiar Bootstrap 5.
- Cambiar arquitectura sin autorización.
- Eliminar funcionalidades existentes.
- Crear código duplicado.

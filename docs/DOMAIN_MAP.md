# Domain Map

## Objetivo

Este documento presenta los dominios funcionales que conforman Reuniones JW y cómo se relacionan entre sí.

No describe la implementación técnica ni la base de datos.

Su objetivo es servir como mapa conceptual del negocio.

---

# Plataforma

Responsable de la administración del sistema.

## Componentes

- Usuarios
- Roles
- Permisos
- Auditoría
- Configuración Global
- Multitenancy

---
